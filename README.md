# 🌌 Млечный Путь — интерактивная 3D-визуализация галактики

[![Three.js](https://img.shields.io/badge/Three.js-r128-black?logo=three.js)](https://threejs.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> Реалистичная модель спиральной галактики с перемычкой, построенная на **Three.js** и **ES6-модулях**.  
> Включает балдж-бар, четыре спиральных рукава, звёздное гало, постобработку Bloom и плавное управление камерой.

---

## ✨ Особенности

- **500 000 звёзд** — генерируются процедурно с реалистичным распределением.
- **Балдж с перемычкой** — эллипсоидная форма с углом наклона, имитирующая центральный бар Млечного Пути.
- **Спиральные рукава** (4 шт.) — логарифмическая спираль с настраиваемым углом закрутки и плавным выходом из бара.
- **Диффузное гало** — редкие звёзды вокруг галактики, придающие глубину.
- **Bloom‑эффект** — свечение ярких звёзд благодаря `UnrealBloomPass`.
- **Управление камерой** — вращение, масштабирование и панорамирование с помощью `OrbitControls`.

---

## 🎮 Управление

| Действие                | Клавиша / Жест              |
|-------------------------|-----------------------------|
| Вращение камеры         | Левая кнопка мыши + движение|
| Приближение / отдаление | Колёсико мыши               |
| Панорамирование         | Правая кнопка мыши + движение|
