# Разработка системы трёхмерной визуализации положения объектов в галактической системе координат

## Аннотация

Добро пожаловать в репозиторий нашего проекта! Наша команда работает над тем,чтобы визуализировать пространственное распределение некоторых молекул в нашей галактике на 3D модели Млечного пути. Поиском данных занимается другая команда - мы лишь визуализируем данные. В данном readme будет представлена краткая характеристика итогового продукта и инструкция по сборке. Будем надеяться, что наша модель вам понравится.

## Языки программирования и вид продукта

Наш итоговый продукт - веб-приложение в котором будет интерактивная модель галактики, а также меню для взаимодействия с моделью. Данные для отображения объектов на диске галактики, предоставляемые взаимодействующей командой, обрабатываются скриптом на Python и представляются в удобном формате. Далее обработанные данные обрабатываются и моделируются с помощью JavaScript и движка для 3D визуализации Three.js. Интерфейс и дизайн приложения написаны с помощью HTML и CSS. 

## Ожидаемые результаты

Ожидается, что итоговый пользователь сможет динамически изменять положение камеры в модели Млечного пути в пространстве, при наведении на объект получать краткую справку, а при клике - полную информацию об объекте и содержании молекул. Концентрация молекул в спектре будет отражаться на модели яркостью света от объекта.  В меню можно будет как посмотреть информацию об объекте,так и выбрать определённые молекулы, которые будут отражены на модели.
